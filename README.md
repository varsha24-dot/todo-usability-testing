# Usability Testing and Analysis of a To-Do List Web Application

## Project Overview

This project focuses on usability testing and analysis of a To-Do List Web Application developed using HTML, CSS, and JavaScript.

The application allows users to:
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks

The main goal of the project is to evaluate user experience, collect feedback, identify usability issues, and improve the interface through iterative design improvements.

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

---

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Responsive design
- Local storage support

---

## Usability Testing Process

The usability testing process included:
- Preparing test scenarios
- Conducting testing sessions with users
- Recording observations
- Collecting user feedback
- Improving the design based on findings
