# Usability Test Plan

## Project Title
Usability Testing and Analysis of a To-Do List Web Application

## Objective
To evaluate the usability, efficiency, and user satisfaction of the To-Do List application.

## Participants
5 users including students and friends.

## Tasks Given to Users

1. Add a new task
2. Mark a task as completed
3. Delete a task
4. Filter completed tasks
5. Filter pending tasks

## Metrics Collected

- Time taken to complete tasks
- Number of errors
- Ease of navigation
- User satisfaction
- Feedback and suggestions

## Testing Environment

- Laptop/Desktop
- Google Chrome Browser
- Windows Operating System
